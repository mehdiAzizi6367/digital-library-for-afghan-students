Digital Library System for afghan students

A modern **Digital Library Management System** built with (Laravel) to manage books, users, and multilingual content efficiently.

---

Overview

This project is designed to provide a simple and powerful platform for managing a digital library with support for multiple languages such as **Pashto, Dari, and English**.

---

## ✨ Features

* 👤 User Management (Admin / Users)
* 📖 Book Management System
* 🌐 Multi-language Support (Pashto 🇦🇫 | Dari 🇦🇫 | English 🇬🇧)
* 📊 Dashboard with Reports
* 🔍 Search & Filter Books
* 📥 Download Tracking
* ⚡ Clean and Responsive UI (Bootstrap)

---

 Technologies Used

* **Backend:** Laravel (PHP)
* **Frontend:** Blade, Bootstrap, JavaScript
* **Database:** MySQL
* **Version Control:** Git & GitHub

## ⚙️ Installation Guide

Follow these steps to run the project locally:

```bash
# Clone the repository
 https://github.com/mehdiAzizi6367/digital-library-for-afghan-students.git

# Go to project folder
cd library

# Install dependencies
composer install

# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Run migrations
php artisan migrate

# Start the server
php artisan serve
```
